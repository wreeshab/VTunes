# VTunes

## Tech Stack

- **Frontend:**

  - React
  - Tailwind CSS
  - Axios
  - React Router

- **Backend:**

  - Node.js
  - Express
  - MongoDB
  - Mongoose

- **Authentication:**

  - OAuth2(Delta Auth)
  - JWT(in-house auth)

- **Real-time Communication:**
  - Socket.IO

#### PORTS

- 3000 -> Client
- 4000 -> Artist-portal
- 5000 -> Server
